## Installation

### Windows

```
pip install virtualenv
virtualenv venv
\venv\Scripts\activate.ps1
pip install -r requirements.txt
python manage.py runserver
```

### Linux and Mac

```
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python manage.py runserver
```
